# Cosentino Urbanismo, Assets para o Site

Pacote curado de assets da marca Cosentino Urbanismo para upload no projeto Lovable do portal `cosentinourbanismo.com.br`. Os arquivos aqui vieram do OneDrive institucional da agência Desigual e foram selecionados e renomeados para uso semântico direto no site.

Ver `MANIFEST.md` para o detalhamento de cada arquivo e em que seção do site ele se encaixa.

## Estrutura

- `logos/` — logos dos 6 empreendimentos Cosentino em vetor (AI, PDF) e raster (PNG)
- `hero/` — aéreas e vídeo hero para a página inicial e landing do Europa V
- `galeria/` — aéreas para o mosaico editorial
- `splendore/` — comunicação visual do Splendore Resort Residence
- `portfolio/` — (vazio, reservado para novos empreendimentos)

## Notas

- Os vídeos originais `.mov` (> 100 MB cada) não estão versionados no GitHub por limite de tamanho. Estão disponíveis localmente no vault ou no OneDrive da agência em `Desigual 2026/Cosentino/JARDIM EUROPA V/`.
- Os MP4 transcodados (H.264 1920x1080, ~10 s, 6–11 MB) estão versionados e prontos para upload no Lovable.
- Logo oficial da marca-mãe "Cosentino Urbanismo" ainda será construído no Claude Design. Prompt pronto em `../lovable-briefs/prompt-logo-cosentino-urbanismo.md`.

## Uso no Lovable

1. Abrir o projeto Lovable da Cosentino Urbanismo
2. Fazer upload em lote dos assets desta pasta
3. Colar o brief de refinamento do arquivo `lovable-briefs/02-home-refinamento-v2.md` no chat do projeto
4. Revisar e aprovar seção por seção
